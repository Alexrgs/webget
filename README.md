#webget

```
webget: get files from web
usage: webget [options]... [url]...

example: webget -e mp3,ogg http://www.astronomycast.com/archive/

options:
  -e, --extensions=LIST                comma-separated list of accepted extensions
  -p, --proxy=[user:pass@]ip[:port]    proxy settings if required
  -u, --user-agent=NAME                value for User-Agent HTTP field, e.g. "Links (0.96; Linux 2.4.20-18.7 i586)"
  -s, --save-directory=PATH            output directory path for downloaded files
  -h, --help                           print this help
```
